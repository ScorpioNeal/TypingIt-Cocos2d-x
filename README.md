最强手指 (Typing It) 
==================


##说明
打字游戏，每打一个字母虚拟键盘的键位就会变换一下。
游戏展示： [Goole Play下载地址][1]

##开发环境
Cocos2d-X 2.2.3

##编译方法
 - 将项目导入Eclipse
 - 编辑proj.android/build_native.sh，设置正确的NDK
